# My First HTML Project

This is my very first web development project built using HTML. I created this project to learn the fundamentals of web design, page structuring, and working with basic HTML elements.

## 🛠️ Technologies Used
* **HTML5** (HyperText Markup Language)
* **Git & GitHub** (for version control)

## 🚀 What I Learned:
* How to structure a web page using HTML tags.
* Creating headings, paragraphs, lists, and links.
* Formatting text and organizing project files.
* Pushing local project files to GitHub.
